# Описание

В работе реализована фильтрация изображения с шумом типа соль и перец на GPU.
Программа загружает входное изображение в градациях серого, применяет медианный фильтр и сохраняет результат в выходной BMP-файл.
Для ускорения обработки используется параллельная архитектура GPU: каждый поток обрабатывает отдельный пиксель изображения независимо от остальных.

Обработка пикселя была выполнена различными фильрами - окрестностью 3x3 и окрестностью 5x5. Такой фильтр был выбран в связи с тем, что окресность 3x3 не давала хороших результатов фильтрации на тестовом изображении. Также для улучшения итогового результата были добавлены итерации - фильтр применялся несколько раз к уже отфильтрованному изображению. Из результатов, можно заметить, что самая лучшая фильтрация, которая удалила почти все шумы на изображении - это фильтрация окрестностью 5x5 с 5 итерациями. Понятно, что при это данный вариант и самый затратный по времени.

# Результаты


| Окно фильтра | Количество итераций | Время, ms |
| --- | --- | --- |
| 3x3 | 1 | 0.625056 |

| 5x5 | 1 | 5.23994 |

# Выводы

Использование GPU позволяет распараллелить обработку пикселей, поскольку вычисление для каждого пикселя не зависит от остальных и может выполняться одновременно.

Экспериментальные результаты показывают, что увеличение размера фильтра и количества итераций приводит к росту времени обработки, но при этом приводит к значительному улучшению качества фильтрации шумов.
